Expand description

A trait representing a raytracing object A raytracing object is a 3-dimensional object that provides methods for computing ray bounces and lighting

fn ray_intersects(&self, r: &Ray, ray_t: Interval) -> Option<HitRecord>

Compute whether the given ray in the specified interval intersects with the object If the ray intersects the object, this method should return a HitRecord with information about the intersection Otherwise, this value should return None
